# wordpress-mysql-docker-compose

## Run
#### Create `.env` file from `.env.sample` and edit you like.
```shell
$ cp .env.sample .env
```

#### Start docker-compose
```shell
$ docker-compose up
```

If you want to run background, use `-d` option.
```shell
$ docker-compose up -d
```

Then visit `http://localhost:8080`
